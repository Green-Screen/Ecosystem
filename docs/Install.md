# Installation
# Current version: 2.4.1

## Method #1 - Roblox
Download straight from the [roblox marketplace](https://create.roblox.com/store/asset) this will only contain the module itself.

## Method #2 - Github
Download straight from the releases subdirectory of this [repository](https://github.com/Green-Screen/Ecosystem/releases)

### Download options

    1. Install the [Ecosystem.V2.4.1WithDependencies.zip](https://github.com/Green-Screen/Ecosystem/releases/download/V2.4.1/Ecosystem.V2.4.1.WithDependencies.rbxm) file for just the runtime module with its dependencies
    2. Install the [Ecosystem.V2.4.1WithTests](https://github.com/Green-Screen/Ecosystem/releases/download/V2.4.1/Ecosystem.V2.4.1.WithTests.rbxm) file for runtime and test scripts
    3. Install the [Ecosystem.V2.4.1](https://github.com/Green-Screen/Ecosystem/releases/download/V2.4.1/Ecosystem.V2.4.1.luau) file WITHOUT dependencies. **[Dependencies](#dependencies) are needed to run**

## Method #3 - Git Clone
Clone this repository straight into your project with Git

### Git steps:

```bash
git clone https://github.com/Green-Screen/Ecosystem.git
```

## Dependencies:
-   [EasyType](https://github.com/Green-Screen/EasyType)
-   [TramService](https://github.com/Green-Screen/TramService)
-   [SecureClasses](https://github.com/Green-Screen/SecureClasses)
-   SmartTable (_No repo_)
